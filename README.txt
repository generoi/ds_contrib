DS contrib
==========

* Prefixed/Suffixed/hyphenated/trimmed variant of node/user/term titles
* Bean titles are formatted just like node/term titles
* Webform wrappers can be formatted
* Recipe is provided with two additional formattable fields: yields and total cooking time.
* Facebook comments have an additional comment count field.
* Rate widget is rendered through DS so it can be formatted and positioned.
* Views titles can be formatted like node/term titles

Installation
------------
1. Enable
2. Reconfigure / Configure your fields.
